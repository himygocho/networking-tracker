# Networker

A personal relationship management application built to help people intentionally invest in the relationships that matter most.

## Overview

Networker is a full-stack web application designed to organize contacts, track conversations, remember important details, and encourage meaningful follow-ups.

The goal is to make networking more personal rather than transactional by helping users remember names, conversations, birthdays, goals, prayer requests, meetings, and other important information about the people they know.

This project also serves as my long-term software engineering portfolio project. Instead of building dozens of unrelated tutorial applications, I am continuously expanding this project as I learn new technologies and improve my development skills.

---

## Current Features (Version 1)

* Add new contacts
* Edit existing contacts
* Delete contacts
* Search contacts
* Store:

  * Name
  * Email
  * Phone Number
  * Notes
